# NERDTree
:NERDTree for tree to open 
q to quit (inside window)



# Tabs
gt for previous tab
gT for next tab
number gt for number of tab (1gt, 2gt..)


# Split window
ctrl+w for starting command after that you can:
    h, j, k, l for moving in windows
    v = vertical split
    h = horizontal


# visual mode
v to get there
y to copy 

p to paste (in command mode)


# Undo / redo 
u = undo
ctrl + r = redo


ctrl + u = up for half page
ctrl + d = down for half page

# Get shell
:term powershell
:term cmd
:tab term powershell
It is hard to get out of it only with the mouse -.-
